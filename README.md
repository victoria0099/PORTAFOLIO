# Personal Portfolio Website

##  Description

This project is a personal portfolio website developed using **HTML, CSS, and JavaScript**. It showcases personal information, projects, and a section dedicated to pets, combining creativity, design, and basic interactivity.

The website reflects a multidisciplinary approach, integrating **technology, art, and personal expression**.

---

##  Features

* Responsive navigation bar with menu links
* Home section with a styled background
* "About Me" section with personal biography
* Projects section with links to research pages
* Pets section with detailed cards and images
* Contact form for user interaction
* Basic JavaScript interactivity (text and color changes)

---

##  Technologies Used

* HTML5
* CSS3 (Flexbox & Grid)
* JavaScript (DOM manipulation)

---

##  How It Works

* The navigation bar allows users to move between sections and pages.
* CSS is used to style the layout, including grids for responsive design.
* JavaScript adds interactivity through buttons that:

  * Change the title text
  * Change text color
  * Reset text and color

The goal of this project is to:
